Bundled logo presets
====================

Drop transparent PNG logos in this folder and they appear as ready-to-use
presets in the editor's Logo tool (the "Saved logos" strip).

These ship with the app (read-only). Logos a user imports with the tool's
"Choose PNG…" button are copied into their own writable folder instead
(%APPDATA%\Manoni\my_logos when built as an exe, or ./my_logos when run from
source), so both sets show up together in the strip.

Only .png files are scanned. Transparency is respected.
